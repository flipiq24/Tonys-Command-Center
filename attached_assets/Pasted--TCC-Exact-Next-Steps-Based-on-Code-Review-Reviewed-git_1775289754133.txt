# ITEM 1: SET ENVIRONMENT TOKENS

The brief route already checks for these. Set them and the live data turns on automatically.

### GMAIL_TOKEN + GOOGLE_CALENDAR_TOKEN

These are OAuth2 access tokens. The brief route uses them with `googleapis`:

```
// brief.ts line 79-82:
function buildGmailClient(token: string) {
  const oauth2 = new google.auth.OAuth2();
  oauth2.setCredentials({ access_token: token });
  return google.gmail({ version: "v1", auth: oauth2 });
}
```

**Problem:** Access tokens expire in 1 hour. You need a refresh token flow.

**Fix — Option A (Quick):** Get a short-lived access token from Google OAuth Playground:
1. Go to https://developers.google.com/oauthplayground/
2. Select Gmail API v1 (all scopes) + Calendar API v3
3. Authorize with tony@flipiq.com
4. Exchange for tokens
5. Set in Replit secrets:
   - GMAIL_TOKEN = the access_token
   - GOOGLE_CALENDAR_TOKEN = same access_token (both are Google APIs)
6. **This will expire in 1 hour.** For production, need Option B.

**Fix — Option B (Production):** Modify `brief.ts` to use refresh tokens:

Replace lines 79-82 in `brief.ts` with:
```typescript
function buildGmailClient() {
  const oauth2 = new google.auth.OAuth2(
    process.env.GOOGLE_CLIENT_ID,
    process.env.GOOGLE_CLIENT_SECRET
  );
  oauth2.setCredentials({ refresh_token: process.env.GOOGLE_REFRESH_TOKEN });
  return google.gmail({ version: "v1", auth: oauth2 });
}
```

Same for `buildCalendarClient()` on line 87.

Then set in Replit secrets:
```
GOOGLE_CLIENT_ID=<from Google Cloud Console>
GOOGLE_CLIENT_SECRET=<from Google Cloud Console>
GOOGLE_REFRESH_TOKEN=<from OAuth playground, step 2>
```

And update the `fetchLiveEmails()` function (line 98) to call `buildGmailClient()` instead of `buildGmailClient(process.env.GMAIL_TOKEN)`.
Same for `fetchLiveCalendar()` (line 167).

# ITEM 2: AUTO-REFRESH

Add this to `artifacts/tcc/src/App.tsx`.

### Find line 83-86 (the clock interval):
```typescript
useEffect(() => {
  const i = setInterval(() => setClock(...), 30000);
  return () => clearInterval(i);
}, []);
```

### Add this NEW useEffect right after it (around line 87):

```typescript
// ═══ AUTO-REFRESH: Pull fresh brief every 15 minutes ═══
const [lastRefresh, setLastRefresh] = useState(
  new Date().toLocaleTimeString("en-US", { hour: "numeric", minute: "2-digit" })
);

useEffect(() => {
  const refreshBrief = async () => {
    try {
      const freshBrief = await get<DailyBrief>("/brief/today");
      if (freshBrief) {
        setBrief(freshBrief);
        setLastRefresh(
          new Date().toLocaleTimeString("en-US", { hour: "numeric", minute: "2-digit" })
        );
      }
    } catch {
      // Silent fail — don't disrupt Tony's workflow
    }
  };

  const interval = setInterval(refreshBrief, 900_000); // 15 minutes
  return () => clearInterval(interval);
}, []);
```

### Pass lastRefresh to Header:

In App.tsx, update the SharedHeader component (around line 240):

```typescript

    lastRefresh={lastRefresh}  // ADD THIS

```

### Update Header component to show it:
